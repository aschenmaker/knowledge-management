# 希尔排序（Shell Sort）

1959年Shell发明，第一个突破O(n2)的排序算法，是简单插入排序的改进版。它与插入排序的不同之处在于，它会优先比较距离较远的元素。希尔排序又叫缩小增量排序。

具体的算法描述过程：

  • 选择增量顺序序列t1，t2，…，tk，其中ti>tj，tk=1；
  • 按增量序列个数k，对序列进行k趟排序
  • 每趟排序，根据对应的增量ti，将待排序列分割成若干个长度为m的子序列，分别对各子表进行直接插入排序。仅当增量因子为1时，当作一个表来处理，此时长度为整个序列的长度

public static void shellSort(int[] arr) {
  int length = arr.length;
  int temp;
  // 分为两个一组，一共有 step/2 组
  for (int step = length / 2; step >= 1; step /= 2) {
    for (int i = step; i < length; i++) {
      temp = arr[i];
      int j = i - step;
      while (j >= 0 && arr[j] > temp) {
        arr[j + step] = arr[j];
        j -= step;
      }
      // 如果没有发生交换，j + step = i，即没有发生交换;
      // 如果发生了交换，则 j + step = i - step, 即下一步进行交换；
      arr[j + step] = temp;
    }
  }
}
